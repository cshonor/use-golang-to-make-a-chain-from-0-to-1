## 🔑 核心概念速览

### UTXO 的本质
- **UTXO = Unspent Transaction Output（未花费交易输出）**
- 就像 "数字现金" 或 "内存碎片"
- 你的余额 = 所有 UTXO 的总和

### Input 和 Output
- **Input** = 你花掉的旧钱（旧 UTXO）
- **Output** = 这次交易产生的新钱（新 UTXO）
- Output 被确认后，就是未来的 Input

### 手续费机制
- **手续费 = Input 总额 - Output 总额**
- 没有专门的 Gas 字段
- 通过 "多输入少输出" 的差额产生

### UTXO 合并
- 把一堆小 UTXO 合成一个大 UTXO
- 就像内存碎片整理
- 可以降低后续交易成本

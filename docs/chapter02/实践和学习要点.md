## 2.8 实践示例

### 2.8.1 完整操作流程

#### 场景：创建账户并查询余额

**步骤 1：创建密钥**
```bash
bytomcli create-key
```

**步骤 2：创建账户**
```bash
bytomcli create-account --quorum 1
```

**步骤 3：创建接收地址**
```bash
bytomcli create-account-receiver --account-id <account-id>
```

**步骤 4：查询余额**
```bash
bytomcli list-balances
```

**步骤 5：查询账户信息**
```bash
bytomcli list-accounts
```

### 2.8.2 交易流程

#### 场景：发送一笔交易

**步骤 1：构建交易**
```bash
bytomcli build-transaction \
  --account-id <account-id> \
  --asset-id <asset-id> \
  --amount 100 \
  --receiver <receiver-address>
```

**步骤 2：签名交易**
```bash
bytomcli sign-transaction --password <password>
```

**步骤 3：提交交易**
```bash
bytomcli submit-transaction
```

**步骤 4：查询交易状态**
```bash
bytomcli get-transaction --tx-id <tx-id>
```

### 2.8.3 使用 dashboard

1. **启动 bytomd**
   ```bash
   bytomd node
   ```

2. **打开浏览器**
   - 访问：`http://localhost:9888`

3. **使用 Web 界面**
   - 创建账户
   - 查看余额
   - 发送交易
   - 查询区块

## 2.9 代码学习要点

**关键点：**
- 使用 Cobra 框架构建命令结构
- 清晰的命令分类和层次

**学习价值：**

**学习价值：**
- 如何实现 RPC 客户端

**关键点：**

**学习价值：**

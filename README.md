# Go 语言公链开发实战

> 基于《Go 语言公链开发实战》的系统化学习项目

## 📚 项目简介

这是一个从零开始用 Go 语言实现公链的学习项目，按照《Go 语言公链开发实战》的学习路径进行开发。

通过本项目的学习，你将：
- 深入理解区块链底层原理
- 掌握 Go 语言在区块链开发中的应用
- 具备独立开发公链核心模块的能力
- 为后续学习 Solidity、DeFi 协议打下坚实基础

## 🎯 学习阶段概览

### 第一阶段：前置基础准备（1-2 周）
- Go 语言核心能力强化
- 区块链核心概念认知

### 第二阶段：公链核心功能实现（3-6 周）
- 基础区块与链的实现
- 交易系统开发
- 共识机制实现（PoW）
- P2P 网络模块开发

### 第三阶段：公链进阶功能（4-8 周）
- 智能合约基础
- 账户与状态管理
- 网络共识优化

### 第四阶段：测试、部署与优化（2-4 周）
- 公链测试
- 部署与运维
- 性能优化

### 第五阶段：实战项目（6-8 周）
- 开发完整的轻量级公链
- 编写文档和测试用例

## 💡 学习建议

### 优先级划分

**🔴 必须吃透（核心模块）**
1. 区块与链的实现
2. 交易系统
3. PoW 共识
4. P2P 网络

**🟡 可以略过（进阶模块）**
1. 智能合约（初期可简化）
2. 状态树（MPT）（可先用简单存储）
3. 轻节点（可先只实现全节点）

### 学习技巧

1. **跟着敲代码，不要光看** - 每一章的代码都要亲手敲一遍
2. **遇到难点先标记** - 不要死磕，先标记下来，等整本书过完再回头攻克
3. **配套资料** - GitHub 上找官方源码或社区讨论
4. **循序渐进** - 先实现基础功能，再逐步完善

## 🎯 核心逻辑

公链开发的核心逻辑：
```
区块生成 → 交易验证 → 共识达成 → 网络同步
```

Go 语言的并发特性是实现这些功能的关键优势。

# 项目文件夹结构说明

根据《Go 语言公链开发实战》14章内容，项目文件夹结构如下：

## 📁 文件夹对应关系

### 第一章：公链设计架构
- `docs/` - 架构设计文档和学习笔记

### 第二章：比原链交互工具
- `cmd/cli/` - 命令行交互工具

### 第三章：比原链核心进程 bytomd
- `cmd/node/` - 节点核心进程

### 第四章：API Server 的实现与原理
- `internal/api/` - API 服务器实现

### 第五章：区块和区块链
- `internal/block/` - 区块相关代码
- `internal/blockchain/` - 区块链相关代码

### 第六章：交易
- `internal/transaction/` - 交易相关代码

### 第七章：比原链智能合约
- `internal/contract/` - 智能合约相关代码

### 第八章：BVM 虚拟机
- `internal/vm/` - 虚拟机相关代码

### 第九章：钱包
- `internal/wallet/` - 钱包相关代码

### 第十章：P2P 网络
- `internal/network/` - P2P 网络相关代码

### 第十一章：数据持久化存储
- `internal/storage/` - 存储相关代码

### 第十二章：PoW 共识机制
- `internal/consensus/pow/` - PoW 共识相关代码

### 第十三章：PoS 共识机制
- `internal/consensus/pos/` - PoS 共识相关代码

### 第十四章：区块链的未来发展
- `docs/` - 相关文档和学习笔记

## 📂 完整目录结构

```
golangchain/
├── cmd/                    # 命令行工具和主程序入口
│   ├── cli/               # 第二章：交互工具
│   └── node/              # 第三章：核心进程
├── internal/              # 内部包（不对外暴露）
│   ├── api/               # 第四章：API Server
│   ├── block/             # 第五章：区块
│   ├── blockchain/        # 第五章：区块链
│   ├── transaction/       # 第六章：交易
│   ├── contract/          # 第七章：智能合约
│   ├── vm/                # 第八章：BVM 虚拟机
│   ├── wallet/            # 第九章：钱包
│   ├── network/           # 第十章：P2P 网络
│   ├── storage/           # 第十一章：数据存储
│   └── consensus/         # 共识机制
│       ├── pow/           # 第十二章：PoW 共识
│       └── pos/           # 第十三章：PoS 共识
├── docs/                  # 文档和学习笔记
│   └── (第一章、第十四章相关文档)
├── go.mod                 # Go 模块定义
├── README.md              # 项目说明
├── 章节学习计划.md        # 学习计划
└── 项目结构说明.md        # 本文件
```

## 🎯 使用说明

按照章节学习计划，在对应的文件夹中实现相应的功能模块。

每个文件夹都是空的，等待你根据学习进度逐步添加代码实现。
